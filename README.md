# ZPSwitchoverCustomizedViewController
本Demo主要介绍了如何切换在一个父视图控制器上的多个自定义的子视图控制器，以及如何避免视图控制器对象已经被销毁了，但它的view还存在的情况。除此之外还介绍了父子视图控制器在屏幕旋转的时候是怎样调用监听屏幕旋转的方法的。
